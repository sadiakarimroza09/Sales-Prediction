# Sales-Prediction
The project began by importing the required libraries and loading the Advertising.csv dataset. The data was explored by checking its structure, missing values, and statistical summary. A correlation heatmap and pair plot were created to understand the relationships between advertising channels and sales.
The advertising budgets (TV, Radio, and Newspaper) were used as input features, while Sales was the target variable. A Linear Regression model was trained using the training dataset and tested on unseen data. The model's performance was evaluated using MAE, MSE, RMSE, and R² Score. Finally, an Actual vs. Predicted Sales plot was generated to visualize the prediction accuracy. This project shows how advertising spending can be used to forecast future sales and support marketing decisions.
